    Gestão de Projetos

       

    Iniciação e governança

    • Acessibilidade no DoD desde o início. NBR 1; Guia UK/BR 2;
    • Personas com necessidades diversas; testes com usuários. NBR 1; Guia UK/BR 2;
    • Em cada entrega: teclado, foco, linguagem clara, formulários, mídias com alternativas. WCAG 2.1.x 3, 2.4.7 4, 1.2.x 5, 3.3.x 6; NBR 1;

    Planejamento

    • Atividades, cronograma e responsáveis de acessibilidade. NBR 1; Guia UK/BR 2;
    • Recursos (pessoas, softwares, cursos) previstos. NBR 1; Guia UK/BR 2;
    • Critérios visando WCAG AA e NBR 17225. WCAG 2.2 (nível AA) 7; NBR 1;
    • Participação ativa de PcD e canal de barreiras. NBR 1; Guia UK/BR 2;

    Testes e Conformidade

    • Registrar achados, priorizar correções e acompanhar. NBR 1; Guia UK/BR 2;
    • Alvo: conformidade WCAG AA / NBR 17225. WCAG 2.2 (AA) 7; NBR 1;
